# The Odin Project - Foundations
## HTML Foundations - Recipes Project
### This project is a collection of web pages with some of my favorite dishes
### In this project, I could practice what I learned during the HTML Foundations section of The Odin Project Curriculum
 - The main skills practiced were creating unordered and ordered lists with the use of `<ul>` and `<ol>` tags.
 - Also the use of the `<img>` tag and the `alt` attribute to add images with a description to the web page, following best practices of accessibility
 - Pracitce the use of paragraphs and the different headings, with `<p>` and `<h1>`, `<h2>` and `<h3>` tags.
 - Practice adding links to the page with the anchor tag and the `href` attribute.
   
The recipes used are all from [Allrecipes](https://www.allrecipes.com/)
